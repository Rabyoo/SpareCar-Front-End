# Project Summary
The SpareCar eShop is a full-stack e-commerce platform designed for selling car spare parts and accessories. Built with React.js and Tailwind CSS for the frontend, and Node.js with Express and MongoDB for the backend, it offers a modern, responsive user experience. The project features a product catalog, shopping cart, multi-step checkout process, user authentication, and an admin dashboard for product management.

# Project Module Description
The project consists of several functional modules:
- **User Authentication**: Handles user login, registration, and role management (admin/customer).
- **Product Management**: Displays products, allows filtering by categories, and provides detailed views for individual products.
- **Shopping Cart**: Manages cart items, allows quantity updates, and calculates totals.
- **Checkout Process**: Collects shipping information and processes orders.
- **Admin Dashboard**: Provides tools for managing products and viewing order statistics.

# File Description Inventory
- **README.md**: Provides an overview of the project and setup instructions.
- **index.html**: The main HTML file that loads the React application.
- **App.tsx**: Main application component that sets up routing and context providers.
- **components/**: Contains reusable UI components like Navbar, Footer, ProductCard, etc.
- **context/**: Contains context providers for managing authentication and cart state.
- **pages/**: Contains individual page components like Home, Products, Cart, Checkout, etc.
- **lib/**: Utility functions and types used throughout the application.

# Technology Stack
- **Frontend**: React.js, TypeScript, Tailwind CSS
- **Backend**: Node.js, Express, MongoDB
- **State Management**: Context API
- **Routing**: React Router
- **Styling**: Tailwind CSS
- **Build Tool**: Vite

# Usage
1. **Install Dependencies**: Run `pnpm install` to install all project dependencies.
2. **Build the Project**: Use `pnpm run build` to build the application for production.
3. **Run the Application**: Use `pnpm run dev` to start the development server.
